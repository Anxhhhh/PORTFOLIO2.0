How to Create a "Cursor Mask Reveal" Effect

This guide explains how to build an interactive effect where moving your cursor acts like a "spotlight" that reveals hidden content underneath a dark background. Since you do not have an image, we will reveal a bright version of your text or background when the user hovers over the screen.

We will use Framer Motion and a CSS property called `mask-image`.

### Step 1: Install Framer Motion (Skip if already installed)
Make sure you have Framer Motion installed in your React project.
Run this in your terminal:
`npm install framer-motion`

---

### Step 2: Create a Mouse Tracking Hook
First, we need to know exactly where the user’s cursor is so the spotlight can follow it. We’ll do this using a React Hook.

Create a new file or add this hook directly into your component file:

```jsx
import { useState, useEffect } from "react";

export const useMousePosition = () => {
  const [mousePosition, setMousePosition] = useState({ x: null, y: null });

  useEffect(() => {
    const updateMousePosition = (e) => {
      setMousePosition({ x: e.clientX, y: e.clientY });
    };

    // Listen for mouse movement
    window.addEventListener("mousemove", updateMousePosition);
    
    // Cleanup the event listener when the component unmounts
    return () => {
      window.removeEventListener("mousemove", updateMousePosition);
    };
  }, []);

  return mousePosition;
};
```

---

### Step 3: Build the Mask Reveal Component
The core idea is to physically layer TWO versions of your website on top of each other. 
1. **The Base Layer** (What you normally see: dark theme, subtle text).
2. **The Reveal Layer** (Vibrant text and background) hidden by `mask-image`.

Here is the code to create the effect:

```jsx
import { useState } from "react";
import { motion } from "framer-motion";
import { useMousePosition } from "./your-path-to-hook"; // <-- Import the hook we made in Step 2

const MaskRevealEffect = () => {
  // 1. Get exact mouse coordinates from our custom hook
  const { x, y } = useMousePosition();
  
  // 2. State to track if the user is hovering over the text 
  const [isHovered, setIsHovered] = useState(false);

  // 3. Size of the Mask/Spotlight. 
  // It grows from a 40px circle to a 400px circle when hovering.
  const maskSize = isHovered ? 400 : 40; 

  return (
    <div className="relative min-h-screen bg-[#0c0c0c] flex items-center justify-center overflow-hidden">
      
      {/* =======================================================
          FRONT LAYER: The normal, dark version that is always visible
          ======================================================= */}
      <div className="absolute inset-0 flex items-center justify-center text-gray-600">
        <h1 className="text-8xl font-black uppercase tracking-tighter">
          Mask Reveal
        </h1>
      </div>


      {/* =======================================================
          REVEAL LAYER: The bright, hidden version.
          The `mask-image` property cuts a circular hole so you 
          only see the part where the mouse is.
          ======================================================= */}
      <motion.div
        className="absolute inset-0 flex items-center justify-center bg-white text-black pointer-events-none"
        
        // This is where the magic happens! We animate the mask 
        // to follow the x and y coordinates of the cursor
        animate={{
          WebkitMaskImage: `radial-gradient(circle at ${x}px ${y}px, black 0%, transparent ${maskSize}px)`,
          maskImage: `radial-gradient(circle at ${x}px ${y}px, black 0%, transparent ${maskSize}px)`,
        }}
        transition={{ type: "tween", ease: "backOut", duration: 0.1 }}
      >
        <h1 
          className="text-8xl font-black uppercase tracking-tighter pointer-events-auto"
          onMouseEnter={() => setIsHovered(true)}
          onMouseLeave={() => setIsHovered(false)}
        >
          Mask Reveal
        </h1>
      </motion.div>

    </div>
  );
};

export default MaskRevealEffect;
```

### How it Works
1. `useMousePosition` tracks the X and Y coordinates of the mouse on the screen.
2. The `maskImage` CSS property creates a `radial-gradient` circle.
3. Everything inside this circle (`black 0%`) makes the **Reveal Layer** visible.
4. Everything outside this circle (`transparent`) keeps the **Reveal Layer** invisible, so you only see the **Base Layer** beneath it.
5. Framer Motion smoothly animates the position of this spotlight based on the mouse's X/Y coordinates.
