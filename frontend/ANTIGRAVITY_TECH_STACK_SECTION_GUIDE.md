## 6) Exact visual direction

Antigravity ko styling ke liye yeh direction do:

### Overall section mood
- dark premium
- cinematic
- crisp
- minimal but rich
- futuristic without becoming flashy
- elegant red-accent continuity with existing Skills cards

### Background
Use the same visual family as the rest of the portfolio:
- base: near-black / charcoal (`#0c0c0c` range)
- optional subtle radial glow
- optional grain / noise overlay very low opacity
- optional faint gradient bloom behind section heading or center of card grid

### Heading style
- small uppercase eyebrow with letter spacing
- thin line or accent bar beside heading
- large refined headline beneath
- supporting paragraph in muted gray
- keep left aligned on desktop if that matches existing rhythm

### Grid style
- 2 columns on small screens
- 3–4 columns on tablets
- 4–6 columns on large screens depending on card size
- consistent spacing; avoid cramped look
- each card should feel intentional, not like a basic template box

### Card style
Cards should be more premium than the reference image:
- rounded corners but not overly soft
- subtle border: `border-white/8` or equivalent
- dark translucent fill or layered black panel
- low ambient inner glow or soft spotlight on hover
- depth through shadow + border + blur layering, not through loud effects
- enough vertical padding so cards feel breathable

### Icon treatment
- centered or top-aligned depending on composition
- icon sizing consistent across brands
- if brand icons vary visually, normalize optical size rather than raw pixel size
- avoid blurry or stretched logos
- prefer SVG / clean vector rendering where possible

### Typography inside cards
- tech name should be highly readable
- optional tiny category label can be muted
- avoid overusing blue text like the reference if it does not match current brand
- use white / gray with subtle red accent only where meaningful

### Hover behavior
Hover should feel premium and subtle:
- card lifts slightly (`translateY(-4px)` to `-8px`)
- border becomes a bit brighter
- faint red or white glow appears
- icon scales slightly
- title becomes brighter
- maybe a soft moving gradient sheen, but only if very subtle

### Motion behavior
- section reveal on scroll can be staggered
- cards can fade + slide upward into view
- hover interaction smooth and springy
- do not over-animate every element
- mobile motion intensity should be reduced
